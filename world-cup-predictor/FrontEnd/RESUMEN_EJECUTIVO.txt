🚀 CÓMO INICIAR EN 3 PASOS
═══════════════════════════════════════════════════════════════════════════════

PASO 1: Configurar Backend
   Editar: FrontEnd/js/config.js

   Cambiar:
   API_BASE_URL: 'http://localhost:8080/api'  ← Tu URL de Spring Boot

PASO 2: Levantar Servidor
   $ cd FrontEnd
   $ python -m http.server 8000

   (O usa PHP, Node.js, etc. Ver INICIO_RAPIDO.txt)

PASO 3: Abrir en Navegador
   http://localhost:8000
